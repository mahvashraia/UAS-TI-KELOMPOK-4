Buku, Pesta, Lebih Banyak Pesta!!! adalah adalah sebuah website 
interaktif yang dirancang sebagai pusat informasi (hub) untuk 
acara dan kehidupan sosial mahasiswa di Universitas Indonesia. 

Project ini bertujuan untuk menyediakan jadwal acara kampus yang 
mudah diakses dengan visual yang menarik. 

Website ini terdiri dari 5 bagian utama yang dapat di-scroll 
(single-page application feel), di mana setiap elemen dirancang 
agar responsif dan interaktif. Website dibangun menggunakan 
Vanilla HTML, CSS, dan JavaScript tanpa framework berat untuk 
memastikan performa yang cepat dan ringan. (jujur pake ai mas)

Fitur :

- Event Calendar
Kalender interactable dengan fungsi mengetahui event di 
tanggal tertentu

- Event List
Menunjukkan semua event di UI

- Social Media
Menunjukkan semua sosial media event organizer di UI
